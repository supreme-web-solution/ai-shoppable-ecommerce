## Pro tips for recording

1. **Pre-load demo data** — products, a published video, and a playlist ready before you hit record.
2. **Use one store** — avoid switching stores mid-demo unless you want to show multi-brand.
3. **Hero moment = Create Video + embed preview** — spend the most time here; that's the wow factor.
4. **Keep mouse movement slow** — pause 2–3 seconds on each screen so viewers can read.
5. **Shorter cut?** — Drop Live Cast and compress Dashboard to ~3 minutes total.
